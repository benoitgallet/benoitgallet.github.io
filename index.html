Postdoctoral Scholar

School of Informatics, Computing and Cyber Systems

Northern Arizona University

Research

Interested in High Performance Computing (HPC), parallel computing, and algorithms performance optimizations.

Working on the development and optimization of parallel applications using GPUs, and leveraging recent hardware features such as Tensor Cores.

Working on heterogeneous algorithms using simultaneously the CPU and GPU, and GPU CUDA and Tensor Cores.

Emphasis on data analysis applications such as similarity searches, kNN, or clustering.
